# АкадемиУм v2 — образовательная платформа-тренажёр

Современный одностраничный сайт с **9 тренажёрами**, системой достижений,
ежедневным челленджем и графиками прогресса.

Открывается двойным кликом по `index.html` — никаких серверов, установок,
сборщиков и интернета не нужно.

---

## 🚀 Быстрый старт

1. Двойной клик по `index.html` — сайт откроется в браузере.
2. Выбери тренажёр на главной (или прими челлендж дня).
3. Решай задачи, копи статистику и достижения.

## ✨ Фишки v2

- 🏆 **Система достижений** — 21 бейдж за прогресс (серии, количество правильных ответов, прохождение всех тренажёров, ежедневный streak)

- 🔔 **Звуки** — синтезированные через Web Audio API (без файлов)
- 🌗 **Тёмная и светлая тема** — переключатель в шапке

- 💡 **Объяснения** — после неправильного ответа показывается правило или подсказка
- 🔄 **Миграция со старой версии** — статистика из v1 подхватывается автоматически
- 📱 **Адаптивная вёрстка** — работает на телефоне и десктопе

## 🔧 Архитектурные решения

### Тренеры как модули
Каждый тренажёр — отдельный JS-файл, регистрирующий себя в глобальном
реестре `Trainers`:
```js
Trainers["math-basic"] = {
  id: "math-basic",
  meta: { icon, iconClass, name, desc },
  init() { /* привязывает обработчики */ },
  start() { /* начинает сессию */ },
  // внутреннее состояние
};
```
Это позволяет легко добавлять новые тренажёры — просто создай новый файл
в `js/trainers/` с тем же интерфейсом.

### Версионированное хранилище
Данные хранятся в `localStorage` под ключом `akademium_v2` с полем `version`.
Старые данные из `akademium_stats_v1` автоматически мигрируют при первом
запуске новой версии.

### Достижения как данные
Каждое достижение — это объект с функцией `test(stats)`, которая проверяет
условие. Это позволяет легко добавлять новые достижения без изменения
основного кода.

### Синтезированные звуки
Никаких аудиофайлов — все звуки генерируются на лету через Web Audio API:
- Правильный ответ — восходящее арпеджио
- Неправильный — нисходящий звук
- Завершение сессии — мажорный аккорд
- Достижение — особая мелодия

## 🛠 Хочешь добавить свой тренажёр?

1. Создай `js/data/mytrainer.js` с массивом задач
2. Создай `js/trainers/mytrainer.js`:
   ```js
   Trainers["mytrainer"] = {
     id: "mytrainer",
     meta: { icon: "🎲", iconClass: "myclass", name: "Мой тренер", desc: "..." },
     init() { /* привязать обработчики */ },
     start() { /* начать сессию */ },
   };
   ```
3. Добавь секцию в `index.html` по аналогии с другими
4. Добавь `<script>` теги для data и trainer файлов перед `app.js`
5. Добавь id в `DEFAULT_STATS.trainers` в `storage.js`
6. Перезагрузи страницу — карточка автоматически появится на главной

## 📊 Что сохраняется

- Статистика по каждому тренажёру (правильные ответы, серии, попытки)
- Общее число правильных ответов и попыток
- Лучшая серия за всё время
- Активность по дням (для графика)
- Серия дней подряд (ежедневный челлендж)
- Все разблокированные достижения (21 шт.)
- Выбранная тема (тёмная/светлая)

Всё хранится локально в браузере, ничего не уходит на сервер.
